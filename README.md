# SQL Sales Data Analysis Assignment

## Objective

Analyze sales data using SQL with filtering, aggregation, and business queries.

## Tasks Performed

* Loaded dataset into SQL Server
* Explored sample data
* Applied WHERE filters
* Used GROUP BY for aggregations
* Sorted results using ORDER BY
* Analyzed top customers and sales categories

## Tools Used

* SQL Server Management Studio (SSMS)
* SQL

## Queries Included

* High sales orders
* Loss making orders
* Total sales by category
* Top customers by sales
